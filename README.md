# Predicción de Ventas con Random Forest

Este proyecto utiliza diferentes modelos de machine learning para predecir las ventas totales en un supermercado.

## Modelos Utilizados:
1. **Regresión Lineal**
2. **Árbol de Decisión**
3. **Random Forest**
